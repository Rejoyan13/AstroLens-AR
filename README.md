# AstroLens-AR
AstroLens AR is a Unity-based AR mobile application that uses marker-based image detection to visualize 3D planets. By scanning planet images from a custom-designed book, users can explore interactive 3D models with zoom, rotation, and detailed information in an engaging augmented reality experience.
